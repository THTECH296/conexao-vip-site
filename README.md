# Conexão VIP — Landing Page

Landing page institucional para a Conexão VIP, provedor de internet fibra óptica
de Teófilo Otoni e do Vale do Jequitinhonha.

## Recursos

- Hero com proposta de valor e prova social (avaliação Google)
- Planos de internet fibra com custo por dia
- Seção de streaming incluso (HBO Max, Disney+, Prime Video, Globoplay, Deezer)
- Serviços adicionais (Móvel 5G, telefone fixo, câmeras)
- App do Cliente com badges das lojas
- Central de atendimento e FAQ
- **Consulta de cobertura por CEP** (integração com a API ViaCEP)
- CTA de contato e botão flutuante de WhatsApp
- Totalmente responsivo, com animações de entrada e navegação com scrollspy

## Tecnologias

- HTML5, CSS3 e JavaScript (sem dependências/build)
- Tipografia: Sora + Manrope (Google Fonts)
- API ViaCEP para verificação de cobertura

## Como usar

Abra o `index.html` no navegador — não há etapa de build.

## Estrutura

```
index.html              Página completa (markup, estilos e scripts)
logo-conexaovip.webp    Logo da marca
modelo.webp             Imagem do hero
logos/                  Logos das marcas parceiras e lojas
```
